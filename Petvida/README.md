# Petvida

Repositório com scripts SQL para o projeto Petvida.

## Estrutura

- `Petvida#1/database/schema.sql`: cria o banco de dados `petvida` e suas 6 tabelas.
- `Petvida#1/database/seed.sql`: insere dados de exemplo para espécies, veterinários, tutores, animais, consultas e pagamentos.
- `Cd completo/completo.sql`: script único com schema e seed juntos.

## Como usar

1. Abra o MySQL Workbench.
2. Execute `Petvida#1/database/schema.sql` para criar o esquema.
3. Execute `Petvida#1/database/seed.sql` para popular os dados.
